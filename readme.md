## c-basic-map
This code implements a basic `map` and `map-double` (basically doubles the values in the array) function.

I suck at C, but here's trying out callbacks (not really; function pointers).

Output:
```
1 -> mapDouble(n) -> 2
2 -> mapDouble(n) -> 4
3 -> mapDouble(n) -> 6
4 -> mapDouble(n) -> 8
5 -> mapDouble(n) -> 10
```

### Running
```bash
make
./main.out
```
